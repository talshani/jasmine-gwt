#jasmine-gwt

Simple GWT wrapper for jasmine testing framework

Look inside the example module...


To run the example:
```Shell
gradlew :example:gwtSuperDev
```

Then navigate to: 
```
http://localhost:9876/jasmine.example.JasmineExample/jasmine/runner.html
```
